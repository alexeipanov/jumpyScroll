Basic usage

Include CSS and JS

<link rel="stylesheet" href="jumpyscroll.css">
<script src="jumpyscroll.min.js"></script>

Set up HTML


<body>
<section>
This is a first section!
</section>
<section>
...
</section>
</body>

Call the plugin

$(document).ready(function() {  
    window.jumpyScroll();  
});
